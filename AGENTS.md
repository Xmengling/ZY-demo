# Agent Instructions

- 后续整理方剂时，默认按照 `note\方剂整理.txt` 中已经形成的结构与写法执行。
- 每个方剂都应尽量整理为以下固定模块，模块标题保持一致：
  - `## 方剂名`
  - `### 组成：`
  - `### 相关条文`
  - `### 方剂名病理`
  - `### 辩证要点`
  - `### 临床症状：`
  - `### 现代疾病：`
  - `### 胡希恕`
  - `### 李冠杰`
  - `### 对比`
  - `### 医案`
- `组成`模块只记录中药药物与剂量；原方煎服法、服用方法、先煎后下等用法不要写入`组成`栏。
- `相关条文`模块按条列出《伤寒论》《金匮要略》《外台》《千金》等原文，并尽量保留条文编号或出处。
- `相关条文`模块也要标重点：对最能体现方证主治、辨证抓手、禁忌边界、类方鉴别的原文短语使用 `[[**重点文字**]]` 标记，便于卡片中红色加粗显示。
- `病理`模块用十二字病理体系归纳，优先使用这些标签：半表、半热、半虚、里热、里寒、里虚、水实、血虚、气实、血实、水虚、水实。每条写成“标签：证据/含义”的形式。
- `辩证要点`模块用短句列出临床抓手，突出主症、兼症、适用边界和禁忌，不只照抄条文。
- `临床症状`模块汇总可见症状，尽量覆盖主症、兼症、舌脉、寒热、饮食、二便、胸腹咽喉等相关表现。
- `现代疾病`模块列出可对应的现代病名或常见临床场景；不要把现代病名等同于方证，仍以方证为核心。
- `胡希恕`、`李冠杰`模块分别包含`讲稿原文：`和`要点总结：`；原文尽量保留关键表达，总结要提炼方药病机、主治边界、常见误解和类方区别。
- `对比`模块选择与本方容易混淆、同类病机或同类主症的方剂，先写“几个以……为主的方子对比：A、B、C”，再用编号列出鉴别要点。
- `医案`模块记录代表性病例，包含患者信息、主诉/症状、舌脉、处方、疗效；医案后必须加`十二字病理分析：`，逐项说明症状如何对应病理标签。
- 整理时语言要偏学习笔记风格：准确、简洁、能用于复习和辨方；保留必要原文，但每个模块都要有自己的归纳。
- 后续整理方剂时，要把我认为最重要的辨证抓手、核心病机、用方边界、类方鉴别点等内容标为重点；在数据库和卡片字段中使用 `[[**重点文字**]]` 语法表示红色加粗。
- 后续整理每个方剂时，所有重要知识点都要主动标红，尤其是症状相关内容：`相关条文`中的主症/兼症/禁忌症状、`病理`证据、`辩证要点`核心抓手、`临床症状`主症、`医案`主诉和舌脉疗效要点，都优先使用 `[[**重点文字**]]` 标记。
- 如果我的资料里没有相关的医案，你则去网络上找一些经方名家的医案，包括胡希恕，李冠杰，冯世纶，黄煌、矢数道明、刘渡舟、大塚敬节等这些老师
- 如果资料不足，允许缺少个别模块，但应优先补齐`组成`、`相关条文`、`病理`、`辩证要点`、`对比`和`医案十二字病理分析`。
- 后续整理方剂后，要把整理后的方剂信息，尤其是`医案`和`十二字病理分析`，同步写入 `web\db\jingfang.sqlite3` 的 `formulas` 表，保持网页数据库可直接读取。
- 后续每次整理/写入单个或少量方剂后，不要自动导出 PDF，以节省时间和上下文；只做数据库写入与必要的轻量检查。等累计接近或凑齐 100 个方剂时，再统一导出 PDF 并检查目录、搜索、页脚、卡片排版是否正常。
- 每次写入或更新方剂数据库后，都要打开 Google Chrome 预览本地网页，优先访问 `http://127.0.0.1:5188`；如果服务未启动，先启动 `web\server.py`。

- 修改方剂卡片「预览」相关逻辑时，默认同步两处：**右侧实时预览**、**PNG 下载导出**。
- 方剂卡片不再保留放大卡片功能。
- PNG 导出按钮分为两种：`下载部分`不包含胡希恕解析和李冠杰解析，医案只显示第一个；`下载`包含全部内容，并尽量与右侧预览图内容一致。
- 输入框中的换行（Enter）应在右侧实时预览和 PNG 下载导出中一致显示。

- 后续每次生成或更新 HTML 文件后，都要自动使用 Google Chrome 打开该 HTML 文件进行预览。
- 在 Windows 环境中优先查找 `chrome.exe`，找不到时再检查常见安装路径：
  - `%ProgramFiles%\Google\Chrome\Application\chrome.exe`
  - `%ProgramFiles(x86)%\Google\Chrome\Application\chrome.exe`
  - `%LocalAppData%\Google\Chrome\Application\chrome.exe`
- 打开本地 HTML 时使用文件的绝对路径，例如 `D:\AI\ZY-Study\note\example.html`。
- 如果由于权限限制需要用户批准启动 Chrome，应直接发起权限请求。
